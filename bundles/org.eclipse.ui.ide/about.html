C++ Standard Template Library

This plugin contains a Java port of the code for the STL heap sorting algorithm. The original code can be found at:

    http://www.sgi.com/tech/stl/stl_heap.h

The java implementation of the code can be found in src.zip in the following files:

  • org\eclipse\ui\internal\views\markers\MarkerSortUtil.java
