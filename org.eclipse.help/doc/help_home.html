Using Eclipse help system

Browse topics in the Contents frame () on the left. Click on topics to display them. Use the Back and Forward buttons to navigate within history of viewed topics.

Searching

To quickly locate topics on particular subject in the entire documentation set, enter a query in the Search field. Click () icon to display Search view. You can narrow scope of the search by selecting only sections that you are interested in.

Synchronizing

After you run a search and find a topic you were looking for, click the Synchronize button () to match the navigation tree up with the current topic. You might also find it useful to synchronize after following in-topic links.

More information

If you cannot find solution to your question in the on-line help, visit eclipse.org web site, read articles, and participate in the eclipse community.
