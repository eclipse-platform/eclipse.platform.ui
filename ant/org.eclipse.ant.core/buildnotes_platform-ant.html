Eclipse 3.4 Ant Build Notes

Enhancements in 3.4

A new application entry point (org.eclispe.ant.ui.antRunner) has been added to the Ant UI plug-in that allows an Ant script to be run. This is similar to the existing application entry point (org.eclipse.ant.core.antRunner) in the Ant Core plug-in, but allows ant scripts that require access to the workbench to execute.
